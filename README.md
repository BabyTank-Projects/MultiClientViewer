# MultiClientViewer

A streamlined tool for managing multiple RuneLite clients simultaneously, making it easier to view and control all your OSRS accounts from one interface.

## Features

- View multiple RuneLite client windows in a single organized interface
- Easy switching between different clients
- Simplified multi-account management
- Lightweight and efficient

## Download

Head to the [Releases](../../releases) page to download the latest version of MultiClientViewer.exe

## Installation

1. Download `MultiClientViewer.exe` from the latest release
2. Run the executable - no installation required!
3. Launch your RuneLite clients and manage them through MultiClientViewer

## Requirements

- Windows OS
- RuneLite installed and configured

## Usage

Simply run MultiClientViewer.exe and you can add and organize your active RuneLite clients for easier management.
